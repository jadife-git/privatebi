# PrivateBI

Persönliches BI-Dashboard mit vollständiger Datenpipeline.

## Stack
- **Backend:** Python, FastAPI, PostgreSQL
- **Frontend:** React, Recharts
- **ETL:** CSV-Import mit Pandas

## Features
- Relationales Datenbankschema (PostgreSQL)
- REST-API mit FastAPI
- CSV-Import als ETL-Strecke
- Interaktives Dashboard mit KPI-Kacheln und Linien-Chart

## Architektur
CSV → Python/Pandas → PostgreSQL → FastAPI → React Dashboard
